# ChsiMock项目介绍

---

当下，网络环境早已沦为学历至上的 “角斗场”—— 知乎杠精们唇枪舌剑，动辄以 “**你什么学历**” 蛮不讲理地否定他人，非理智吐槽远胜理性探讨，仿佛无学信网凭证，连发言资格都形同虚设。更啼笑皆非的是，现实中学历早已卷得名不副实，毕业即失业的焦虑挥之不去，网络上却仍将其奉为圭臬，仿佛文凭含金量全凭键盘认证。

为此，我隆重推出这款学信网网页生成工具：一键操作即可量身定制 “杠精认证款” 学历页面，应对 “非名校无话语权” 的无理诘问、规避恶意标签皆游刃有余。毕竟在人均 “985 硕博” 的虚拟赛场，一张无懈可击的学信网截图，反倒比真实能力更能堵住杠精之口 —— 他们只认纸片，不识价值，而我们不过是帮你适配这份本末倒置的荒诞。

## 修改内容（手动配置config.json）

### 1. 学籍信息区域（绿色卡片）

- **标题**: "学籍信息 (计数)"
- **位置**: 页面上方
- **样式**: `xj list-card` (绿色)
- **计数**: 始终显示，如 "学籍信息 (2)"

### 2. 学历信息区域（蓝色卡片）

- **标题**: 
  - 有学历时: "学历信息 (计数)" 
  - 无学历时: "学历信息" (不显示计数)
- **位置**: 学籍信息下方
- **样式**: `xl list-card` (蓝色)
- **占位符**: 
  - 无学历记录时显示: "没有找到您的学历信息" + 帮助链接
  - 有学历记录时移除占位符，显示蓝色卡片

## 数据配置示例

```json
{
  "records": [
    {
      "id": "xj001",
      "type": "xj",
      "studentName": "张三",
      "school": "示例大学",
      "level": "本科",
      "major": "计算机科学与技术",
      "description": "2021级 | 在读"
    },
    {
      "id": "xl001", 
      "type": "xl",
      "studentName": "张三",
      "school": "示例高中",
      "level": "高中",
      "major": "理科",
      "description": "2018级 | 已毕业"
    }
  ]
}
```

## 显示效果

### 场景1: 有2个学籍，有1个学历

```
学籍信息 (2)
├─ [绿色卡片] 示例大学 - 本科
└─ [绿色卡片] 第二学院 - 专科

学历信息 (1)
└─ [蓝色卡片] 示例高中 - 高中
```

### 场景2: 有学籍，无学历

```
学籍信息 (2)
├─ [绿色卡片] 示例大学 - 本科
└─ [绿色卡片] 第二学院 - 专科

学历信息
没有找到您的学历信息
├─ 有学历但是没显示出来
├─ 确保您实名注册时提供的姓名和证件号码...
└─ 若不能确定学校注册信息...
```

### 场景3: 有学历，无学籍

```
学籍信息 (0)
暂无学籍记录

学历信息 (2)
├─ [蓝色卡片] 大学 - 本科
└─ [蓝色卡片] 高中 - 高中
```

## Usage:

1. **启动服务器**:
   
   ```shell
   nohup java -jar *.jar > *.log 2>&1 &
   ```

2. **添加测试数据**:
   
   - 访问 http://localhost:portnum/admin
   - 添加学籍记录（type=xj）
   - 添加学历记录（type=xl）

3. **查看效果**:
   
   - 访问 http://localhost:portnum/
   - 验证学籍显示为绿色卡片
   - 验证学历显示为蓝色卡片
   - 验证计数正确显示

4. **边界测试**:
   
   - 删除所有学历记录，验证占位符显示
   - 删除所有学籍记录，验证占位符显示
   - 各添加一条记录，验证计数为 (1)
