# 广告背景图片说明

本目录用于存放广告栏的背景图片。

## 当前状态
- 系统已配置了**CSS渐变背景作为后备方案**
- 即使没有图片文件,广告栏也能正常显示渐变背景
- 如果需要使用官方原版图片,请参考下方说明

## 可选:获取官方图片

如果想要使用学信网官方的广告背景图片,可以通过以下方式获取:

### 方法1: 从浏览器开发者工具下载
1. 访问官方学信档案页面: https://my.chsi.com.cn/archive/index.jsp
2. 打开浏览器开发者工具 (F12)
3. 切换到 Network 标签
4. 刷新页面,筛选图片类型
5. 找到以下文件并保存到本目录:
   - `kzxz_bg.png` - 考证选择广告背景
   - `zcsx_bg.png` - 在线申请广告背景
   - 其他 `*_bg.png` 文件(如需要)

### 方法2: 直接下载链接
从学信网CDN直接下载对应图片,保存到本目录即可。

## 支持的广告类型

当前CSS支持以下广告类型:
- `vote_bg_kzxz` - 考证选择 (已配置紫色渐变后备)
- `vote_bg_zcsx` - 在线申请 (已配置粉红渐变后备)
- `vote_bg_jygz` - 就业跟踪
- `vote_bg_zxs` - 在线申请
- `vote_bg_jyjz` - 教育经历
- `vote_bg_cyfw` - 创业服务
- `vote_bg_jyqw` - 就业求职
- `vote_bg_xzcp` - 选择产品
- `vote_bg_zytj` - 志愿填报
- `vote_bg_zymyd` - 专业满意度
- `vote_bg_yxmyd` - 院校满意度
- `vote_bg_shgldc` - 社会管理

## 注意事项
1. 图片文件名必须与CSS中引用的名称完全一致
2. 建议使用PNG格式以支持透明背景
3. 推荐图片尺寸: 宽度 750px 左右(适配移动端)
4. 如果不添加图片,系统会自动使用渐变色背景,不影响功能
